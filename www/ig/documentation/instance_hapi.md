## Déployer une instance HAPI

1. Installer docker

https://docs.docker.com/engine/install/

1. Installer l’image docker HAPI

https://hub.docker.com/r/hapiproject/hapi

Sur mac ou linux : `docker pull hapiproject/hapi`

1. Lancer le docker container de hapi en appuyant sur le bouton start

![](https://user-images.githubusercontent.com/48218773/185607031-45268753-0fb3-4e09-9788-0739d29dea02.png)

1. Chercher l’url d’accès à hapi

![](https://user-images.githubusercontent.com/48218773/185607155-189def86-cb2a-4a3f-83b4-ca945077bfa2.png)

1. Vous pouvez accéder à votre instance d’HAPI déployée en local!

![](https://user-images.githubusercontent.com/48218773/185607295-45dc46e9-01c8-4090-83d8-dbd63a83ac5e.png)
