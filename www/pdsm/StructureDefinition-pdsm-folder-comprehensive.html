Resource Profile: PDSm Folder Comprehensive

Official URL: https://interop.esante.gouv.fr/ig/fhir/pdsm/StructureDefinition/pdsm-folder-comprehensive Version: 3.0.0
Active as of 2023-03-15 Computable Name: PDSm_FolderComprehensive

Profil spécifique dérivé du profil IHE MHD v4.0.1 « Comprehensive Folder » créé pour le volet ANS “Partage de documents de santé en mobilité” ; ce profil concerne le classeur.

Usage:

  • Use this Resource Profile: PDSm Comprehensive Provide Document Bundle and PDSm Find Lists Response
  • Refer to this Resource Profile: PDSm SubmissionSet Comprehensive

​ List 0..*FolderComprehensiveA list is a curated collection of resources
​ contained 1..*FrPatientInformation about an individual or animal receiving health care services
​ extension:designationType 1..1DesignationTypeReprésente les codes spécifiant le type d’activité de l’événement clinique ayant abouti au regroupement de la fiche d’un document dans ce classeur.

​ identifier:entryUUID S0..*EntryUUIDIdentifierIdentifiant attribué par le système cible.
​ status S1..1codeReprésente la pertinence de la version d’un classeur.
​ mode 1..1codeMode de traitement qui s’applique au classeur.
​ title 1..1stringReprésente le titre du classeur.
​ code 1..1CodeableConceptDésigne le cas d’utilisation de la ressource List. Il s’agit ici du classeur.
​ subject 1..1Reference(FrPatient) {c}If all resources have the same subject

​ note 0..1AnnotationReprésente les commentaires d’un classeur.
​ entry 1..*BackboneElementListe des fiches composant le classeur.
